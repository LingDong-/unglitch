svg2polylines

Turn svg to plain old polylines for copy pasting into your drawing code

paste svg below

it can be a full svg, or url to svg, or datauri, or svg path, or svg path 'd' attribute

# samples

number of sample points (per polyline)

sample phase

(0.0~1.0) rotate the samples. you'll get 1 additional point for non-0 non-1 values

bounding box

proportionally resize graphics to fit in this box: xmin,ymin,xmax,ymax

translation

additional translation for fine tuning position: x,y

curvature-aware downsample epsilon

maximum distance any point can deviate from its true position. leave empty or '0' for no downsample. make sure #samples is high before using this box

new sample count(s): 

round decimal points

leave empty for no rounding
